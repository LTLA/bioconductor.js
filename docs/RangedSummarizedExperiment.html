RangedSummarizedExperiment

RangedSummarizedExperiment

A RangedSummarizedExperiment is a SummarizedExperiment subclass where each row represents a genomic interval. As such, it stores an additional GRanges of length equal to the number of rows, where each element represents the genomic range for the corresponding row of the SummarizedExperiment. It supports the same set of generics as the SummarizedExperiment.
